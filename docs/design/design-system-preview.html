Transitions & Animations

Transitions provide visual feedback to users when elements change state, creating a more polished and responsive interface. All interactive elements should use our standard transition timing.

Standard Transitions

Our default transition duration is 150ms with an ease-in-out timing function. This provides a balance between responsiveness and smoothness.

<!-- In HTML -->
<button class="transition-all duration-150 ease-in-out hover:bg-amber-500 hover:-translate-y-0.5 hover:shadow-md">
  Button with Default Transition
</button>

/* In CSS */
.interactive-element {
  transition: all 150ms cubic-bezier(0.4, 0, 0.2, 1);
}
.interactive-element:hover {
  background-color: #F59E0B;
  transform: translateY(-3px);

Focus States

Focus states are crucial for accessibility, especially for keyboard users. All interactive elements should have visible focus indicators.

Try it: Click any button, then press Tab to see the focus indicator move between elements.

/* Standard focus styles */
.interactive-element:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px rgba(245, 158, 11, 0.4);
}

/* Alternative with outline */
.interactive-element:focus-visible {
  outline: 2px solid #F59E0B;
  outline-offset: 2px;
}

Accessibility Considerations

Our design system prioritizes accessibility to ensure all users can interact with our interfaces effectively, regardless of their abilities or the devices they use.

Touch Targets

Interactive elements should be large enough to be easily tapped on touch devices. The minimum recommended size is 44×44 pixels.

For touch interfaces, ensure that:

  • Buttons and interactive elements are at least 44×44 pixels
  • Links within text have sufficient spacing between them
  • Adequate spacing between interactive elements (minimum 8px)
  • Form controls have sufficient size for easy interaction
/* In Tailwind CSS */
.touch-target {
    @apply min-h-[44px] min-w-[44px]; /* Minimum touch target size */
    @apply p-2; /* Additional padding for better usability */
}

/* For groups of controls */
.touch-target-group {
    @apply space-x-2 space-y-2; /* Minimum spacing between items */
}

Guidelines for color contrast:

  • Normal text: minimum contrast ratio of 4.5:1
  • Large text: minimum contrast ratio of 3:1
  • UI components and graphic elements: minimum contrast ratio of 3:1
  • Our primary amber color (#F59E0B) with dark text has a contrast ratio of 7:1
  • Our secondary gray color (#6B7280) with white text has a contrast ratio of 4.5:1
